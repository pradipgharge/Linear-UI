Introduction

Linear UI is a component library which has a collection of custom UI components that come in a wide range of sizes, shapes and colors, with the help of these components you can easily build websites and webapps without worrying much about the design part (a problem that developers usually face).



Installation Guide

To use Linear UI components all you need to do is just copy & paste the below code in the <head> tag of your html document, and you are good to go. 

                            
                                <link rel="stylesheet" href="https://linearui.netlify.app/css/main.css">
