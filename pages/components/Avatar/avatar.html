Avatar

Avatars can be used to show user's profile picture on profile information pages, on navigation bars, in blogs etc. Avatars come in different shapes and sizes across the web.

To use any avatar component add the class avatar, followed by the variant class name.

Profile Avatars

Variants available: avatar-xxl, avatar-xl, avatar-lg, avatar-md, avatar-sm & avatar-xs.
