Image

Responsive images are used to display images on the web. Images can be of different shapes such as squre and round shaped, also they can expand to fit parent contianer's width.

Responsive Image

Variants available: To use the responsive image component add the class img-responsive to img tag. 

                            
                                <img class="img-responsive" src="/Assets/Images/responsive-image.jpg" alt="image">

Round Image

To use the round images add the class img-round to img tag, followed by the variant classes.
Variants available: img-xl, img-lg & img-sm 

                            
                                <img class="img-round img-xl" src="/Assets/Images/cat-img-2.jpeg" alt="round image">
                                <img class="img-round img-lg" src="/Assets/Images/dog-img.jpeg" alt="round image">
                                <img class="img-round img-sm" src="/Assets/Images/dog&cat-img.jpeg" alt="round image">

Square Image

To use the square images add the class img-square to img tag, followed by the variant classes.
Variants available: img-xl, img-lg & img-sm 

                            
                                <img class="img-square img-xl" src="/Assets/Images/cat-img-2.jpeg" alt="square image">
                                <img class="img-square img-lg" src="/Assets/Images/dog-img.jpeg" alt="square image">
                                <img class="img-square img-sm" src="/Assets/Images/dog&cat-img.jpeg" alt="square image">
