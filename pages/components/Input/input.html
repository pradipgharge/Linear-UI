Input

Inputs are way of user's interaction with the website, they are used to get information from the user such as login/signup process and in form elements.

To use any input element add the class input, followed by the variant class names.

Standard Inputs

Variants available: std-input 

                            
                                <div class="std-input">
                                    <label for="user-name">Username:</label>
                                    <input id="user-name" placeholder="Enter your username" type="text"/>
                                </div>
                                <div class="std-input">
                                    <label for="password">Password:</label>
                                    <input id="password" placeholder="Enter your password" type="password"/>
                                </div>
                                <div class="std-input">
                                    <label for="mobile-number">Mobile Number:</label>
                                    <input id="mobile-number" placeholder="Enter your mobile number" type="number"/>
                                </div>
                                <div class="std-input">
                                    <label for="user-email">Email id:</label>
                                    <input id="user-email" placeholder="Enter your email address" type="email"/>
                                </div>

Validation Inputs

These type of inputs are generally used in form elements to validate user's response.
Variants available: input-success & input-error

Success ! email matched
Incorrect password ! 
                            
                                <div class="std-input">
                                    <label for="user-name">Username:</label>
                                    <input id="user-name" class="input-success" placeholder="Enter your username" type="text"/>
                                    <span class="input-msg">Success ! email matched</span>
                                </div>
                                <div class="std-input">
                                    <label for="password">Password:</label>
                                    <input id="password" class="input-error" placeholder="Enter your password" type="password"/>
                                    <span class="input-msg">Incorrect password !</span>
                                </div>

Radio Inputs

Radio buttons allow user to select one out of the given choices. It comes with a default choice marked as checked. 

                            
                                <div class="radio-input">
                                    <input id="physics" type="radio" name="subject" value="physics" checked>
                                    <label for="physics">Physics</label>
                                </div>
                                <div class="radio-input">
                                    <input id="chemistry" type="radio" name="subject" value="chemistry">
                                    <label for="chemistry">Chemistry</label>
                                </div>
                                <div class="radio-input">
                                    <input id="maths" type="radio" name="subject" value="maths">
                                    <label for="maths">Maths</label>
                                </div>
                                <div class="radio-input">
                                    <input id="biology" type="radio" name="subject" value="biology" disabled>
                                    <label for="biology">Biology</label>
                                </div>

Checkbox Inputs

Checkbox inputs allow users to select multiple inputs from the given choices. 

                            
                                <div class="checkbox-input">
                                    <label for="english">
                                        <input id="english" type="checkbox" name="languages" value="English" checked>
                                        English
                                    </label>
                                </div>
                                <div class="checkbox-input">
                                    <label for="hindi">
                                        <input id="hindi" type="checkbox" name="languages" value="Hindi">
                                        Hindi
                                    </label>
                                </div>
                                <div class="checkbox-input">
                                    <label for="marathi">

                                        Marathi
