Badge

Badges are used to highlight a user's status or an item's count for quick recognition.

To use any badge component add the class badge, followed by the variant class names.

Badge on avatars

Variants available: bd-xl, bd-lg, bd-md & bd-sm 

                            
                                <div class="badge avatar bd-xl" >
                                    <img class="avatar-img" src="/Assets/images/Avatar-01.svg" alt="avatar">
                                    <div class="bd-status bd-online"></div>
                                </div>
