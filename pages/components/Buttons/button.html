Button

Badges are used to highlight a user's status or an item's count for quick recognition.

To use any badge component add the class badge, followed by the variant class names.

Solid Buttons

Variants available: btn-solid-primary, btn-solid-secondary, btn-solid-danger & btn-solid-disabled 

                            
                                <button class="btn btn-solid-primary">Primary</button>
                                <button class="btn btn-solid-secondary">Secondary</button>
                                <button class="btn btn-solid-danger">Danger</button>
                                <button class="btn btn-solid-disabled">Disabled</button>

Outline Buttons

Variants available: btn-outline-primary, btn-outline-secondary, btn-outline-danger & btn-outline-disabled 

                            
                                <button class="btn btn-outline-primary">Primary</button>
